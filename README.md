# 文本分析与词云生成项目

## 项目简介
本项目通过爬取目标网页内容，对其进行中文分词和词频统计，并生成词云图与词频统计图表，以可视化文本数据。

---

## 功能说明
- **网页内容爬取**：利用 `requests` 获取网页文本并保存到本地。
- **数据清洗与分词**：
  - 使用正则表达式清理非中文字符。
  - 使用 `jieba` 进行中文分词。
- **词云生成**：
  - 根据词频数据生成中文词云，支持设置背景蒙版。
- **词频统计图**：绘制词频前十的词汇柱状图。
